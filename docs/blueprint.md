# **App Name**: ScreenSense

## Core Features:

- Admin Dashboard: A secure web interface for administrators to log in, manage content, configure display settings, and review data.
- Content Upload & Management: Admins can upload various media (images, videos), create text announcements, and store them securely for display.
- Content Playlisting: Admins can organize uploaded content into sequential playlists, setting display durations for each item.
- Digital Signage Display: A full-screen client application that fetches and continuously loops through assigned content playlists and dynamic information.
- Dynamic Information Module: Displays real-time elements such as local weather, current time, and a configurable scrolling announcement ticker, with customizable location-based timing.
- AI-Generated Educational Prompts Tool: A tool that allows administrators to generate concise, engaging science or math-related facts or questions tailored for the display schedule.
- Real-time Synchronization: Admins and screens connect via a cloud-based service for real-time content updates and settings synchronization.
- External Media & Content Integration: Enables uploading media from various sources including devices, YouTube, and other external content providers.
- Advanced Display Customization: Comprehensive settings for display mode (landscape/portrait), per-slide duration, fast-selectable layout templates, faculty-specific content panels, and automatic theme scheduling (e.g., morning/night themes).
- Admin Security & Access Control: Implements enhanced security features such as PIN protection for administrative access and sensitive operations.
- Offline Mode & Data Caching: Supports continuous display operation with local data caching, ensuring content availability even without an internet connection.
- Emergency Broadcast System: Facilitates the immediate broadcast of emergency alerts and critical notifications across all connected digital signage displays.
- Analytics & Performance Monitoring: Provides content usage analytics, application performance monitoring (crash & log reporting), admin activity logging, and real-time device status detection (online/offline).
- Third-Party API & IoT Integration: Seamlessly integrates with academic calendars, external APIs (e.g., news, transportation schedules, campus information), QR codes for supplementary information, and IoT systems for environmental automation.
- Interactive & Experiential Features: Supports interactive widgets (polling, feedback, quizzes), a gamification system to boost user engagement, touchscreen interactive mode, voice command system for hands-free control, and simple AR/VR support for learning materials.
- Content Versioning & A/B Testing: Features for tracking content changes with version history and A/B testing capabilities to optimize content effectiveness.
- Scalable Infrastructure & Management: Includes an automatic backup and restore system, a modular plugin system for future expansions, white-label branding, disaster recovery (failover server), a centralized management dashboard for multi-location control, and synchronous multi-screen display for video walls.
- Mobile Notification Integration: Enables push notifications to mobile applications for administrators and relevant stakeholders regarding system updates or alerts.
- AI-Powered Content Intelligence Tool: Utilizes AI-powered tools for contextual content recommendations and predictive analytics to enhance content delivery, audience engagement, and overall display effectiveness.
- Live Streaming & Internal Broadcasting: Allows for the integration and display of live video streams or internal broadcasts on digital signage.
- Granular Device Control: Provides administrators with detailed, schedule-based control and content assignment capabilities for individual devices or groups of devices.
- Multi-language Support: Offers comprehensive support for multiple languages, including Indonesian, Chinese, and English, across both the administrative interface and display content.
- Attendance/Presence System Integration: Optional integration with attendance or presence systems, with a separate menu for enablement/disablement.

## Style Guidelines:

- Primary color: A balanced and clear blue (#266EB8), selected for its professional and intellectual association, ideal for conveying information and clarity.
- Background color: A very light, desaturated blue (#ECF0F2), providing a clean, open canvas that supports legibility across all content types in a light color scheme.
- Accent color: A vibrant, clear aqua (#4ED6D6) with strong contrast to the primary, chosen to highlight interactive elements and critical information.
- Headline and body font: 'Inter' (sans-serif) for its modern, highly readable, and versatile design, ensuring clear presentation of all information types.
- Utilize a consistent set of clean, minimalist vector icons (e.g., Material Design) that are easily understandable and enhance user experience without clutter.
- Implement responsive, modular layouts for both the admin dashboard and display client, ensuring optimal viewing and management on various screen sizes and orientations.
- Incorporate smooth, subtle transitions between different content items on the signage display, and delicate loading animations for dashboard interactions, improving user experience without being distracting.
