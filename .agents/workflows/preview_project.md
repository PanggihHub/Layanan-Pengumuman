---
description: How to start the preview server for the ScreenSense project
---
This workflow describes how to run the preview dev server for the ScreenSense Next.js application.

1. Ensure all dependencies are already installed (`npm install`).
2. Run the Next.js development server:
```bash
npm run dev
```
3. Open a browser and navigate to `http://localhost:9002` to preview the web application.
